1. Purpose

To verify that Simject-030 Epinephrine Auto-Injector (Adult) project product performance meets requirements.

2. Scope

This protocol is applicable to Simject-030 Epinephrine Auto-Injector (Adult) project functional test verification.

3. Responsibilities

Analyst: Responsible for drafting this protocol, completing testing according to the protocol, and issuing test reports.

R&D Engineer: Responsible for reviewing this protocol.

R&D Director: Responsible for approving this protocol and verification reports.

7.3 Test Environment Requirements

Temperature: 23 ± 5°C and Relative Humidity: 50 ± 25% RH

7.4 Auto-Injector Pre-test Conditioning

Before testing, condition at standard temperature: 23 ± 5°C and relative humidity: 50 ± 25% RH for at least 4 hours.

7.5 Auto-Injector Assembly

Assemble conditioned front and rear components with batch XX pre-filled syringe (PFS) according to "000-000-TIS-001 Lisa & Lucy Manual Final Assembly Fixture Test Instruction V1.0" to obtain finished auto-injectors.

7.6 Test Method

Conduct functional testing according to "100-005-TIS-023 V1.0 Final Assembly" for three independent batches, with 10 samples per batch.

7.7 Test Items and Acceptance Criteria

No. Test Item Acceptance Criteria
1 Fill 1 ± 0.03 mL, single injection 0.3 mL (±5%); 0.27 < injection volume < 0.34 mL
2 Needle extension length (during injection): ≥0.5 in (12.70 mm) 0.6 in (±0.1 in) / 15.24 mm (±2.54 mm)
3 Cap removal force: 5-35 N 5-35 N
4 Injection activation force (pressing needle shield): 5-25 N 5-25 N
5 Injection time for 0.3 mL dose: ≤2 seconds ≤2 seconds
6 Drug visibility before use for inspection Viewing window clearly identifiable
7 Post-injection needle shield lockout force >2× injection activation force >50 N
8 User ability to distinguish device status (unused/in use/used/disabled) After injection, PFS stopper visible at window
9 Needle shielding When contacting test surface, needle not visually visible; when not contacting, needle does not protrude
10 Injection completion indication (visual/auditory/tactile or combination) Injection start audible prompt, measured at 30 cm from device, loudness ≥60 dB
11 Sharps injury protection ISO 23908-2024 appendix test method

8. Verification Report

8.1.1. The report shall contain detailed information of the auto-injector and PFS used, including model, batch number, quantity, etc.

8.1.2. The report shall contain functional test data.

8.1.3. The report shall include result determination. If testing fails, re-testing shall be conducted until pass. Each testing process and result shall be recorded and described in detail in the report.
