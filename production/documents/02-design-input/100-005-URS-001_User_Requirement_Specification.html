2. Purpose

To support specific user requirements for a disposable auto-injector for epinephrine injection (pre-filled), this document describes the functionality and performance that the product should have.

3. Scope

Zhejiang Summed Medtech Co., Ltd. shall execute product design, development, verification, and manufacturing according to the requirements of this document.

5. Intended Use

Used in conjunction with epinephrine injection (pre-filled) for fluid delivery (intramuscular injection). For single patient or individual single use only.

  • Adult Dose: 0.30 mg/0.3 mL (1.0 mg/1.0 mL concentration)
  • Pediatric Dose: 0.15 mg/0.3 mL (0.5 mg/1.0 mL concentration)
  • Fill Volume: 1 ± 0.03 mL
  • Injection Volume: 0.3 mL (0.27-0.34 mL)

6. Target Patient Population

Target Population: Adult model suitable for patients with body weight ≥30 kg. Pediatric model suitable for patients with body weight 15-30 kg.

8. Drug and Inner Package Information

Item Requirements
Product Name Epinephrine Injection (Pre-filled)
Description Colorless or almost colorless clear liquid, deteriorates easily when exposed to sunlight or air. Store in light-resistant, tightly closed container at 20-25°C
Therapeutic Area Emergency allergic reactions (anaphylactic shock, asthma), cardiac arrest, etc.
Adult Dose 0.30 mg/0.3 mL (1.0 mg/1.0 mL), fill 1 ± 0.03 mL, single injection 0.3 mL (0.27-0.34 mL)
Pediatric Dose 0.15 mg/0.3 mL (0.5 mg/1.0 mL), fill 1 ± 0.03 mL, single injection 0.3 mL (0.27-0.34 mL)
Needle Extension Adult: ≥0.5in (12.70mm)
Pediatric: ≥0.4in (10.16mm)
Kinematic Viscosity 1.12 cP (test with water for injection: 1.04 cP)
Test condition: Liquid temperature 24.8°C
Density Adult (0.30 mg/0.3 mL): 1.0016 g/mL
Pediatric (0.15 mg/0.3 mL): 1.0009 g/mL
