## üö® Troubleshooting

### **Issue**: Text too small on TV
**Solution**:
- Increase browser zoom: `Ctrl + +` (Windows) or `Cmd + +` (Mac)
- OR decrease TV resolution to 1920√ó1080 (if using 4K)

### **Issue**: Tree not expanding/collapsing
**Solution**:
- Refresh page (F5)
- Check JavaScript is enabled
- Try different browser (Chrome/Edge recommended)

### **Issue**: Search not working
**Solution**:
- Clear browser cache (Ctrl+Shift+Delete)
- Ensure you're in BOM Tree view (View 2)
- Check spelling of search term

### **Issue**: Tables cut off on screen
**Solution**:
- Decrease browser zoom: `Ctrl + -` (Windows) or `Cmd + -` (Mac)
- Use horizontal scroll (touchpad/arrow keys)
- Switch to fullscreen mode (F11)
