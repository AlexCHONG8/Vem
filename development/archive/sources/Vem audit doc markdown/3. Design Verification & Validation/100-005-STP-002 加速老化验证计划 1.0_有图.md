# 1.目的

验证新博研桥专案产品在规定的储存条件下，在有效期内保持其性能符合预期要求。

# 2.范围

适用于新博研桥专案产品加速老化评价，包括加速老化和产品性能验证。

# 3.货架寿命和储存条件

储存条件：温度：0-40℃；湿度35%-75%RH

拟定申报有效期：30M

# 4. 测试要求

# 4.1环境条件

实验室的温度和湿度必须控制在  $25 \pm 3^{\circ} \mathrm{C}$  和  $55 \pm 20 \%$  RH 范围内；

所有测试设备必须在校准期内。

# 8. 测试项目与测试方法

# (1) 自动注射笔组装

将调节好的前组件和后组件，与批号 XX 的预灌封注射器（PFS），按照《000-000-TIS-001 Simject 手动终端组装工装测试指导 V1.0》的指导进行组装，得到成品自动注射笔。

# (2) 产品按照上述要求进行加速条件处理后，应按下表要求进行产品性能测试
