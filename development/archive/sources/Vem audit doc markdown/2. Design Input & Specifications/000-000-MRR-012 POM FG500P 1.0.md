# 避免接触的材料

不相容于强酸和碱(分解形成甲醛)和强氧化剂。在熔化温度下，乙缩醛树脂与卤代聚合物，如PVC和PVDC和任何含有卤代聚合物的高弹性体是不相容的。甚至像污染物那样微小的量，也能引起突然的和自发性的甲醛气体生成。工作场所的烟浓度有可能远高于阈值水平。会导致设备如挤出机，模具的不安全增压。不要污染原生树脂或再造。请勿将本树脂与本公司指定以外的颜料或添加剂混合使用。在没有咨询公司的情况下，不要将该等级与其他等级的缩醛混合使用，也不要与其他树脂混合使用。做任何以上的情况，可能会改变这种树脂的热稳定性，并可能导致分解。

# 有害分解产物

：此物质的分解取决于它暴露于升高的温度以及压力中的时间长短。

在推荐的加工温度下，在30分钟内分解应不会很明显。

由于污染物，色素和/或其他的添加剂，分解可能会加速。

由于有压力的蒸汽，在高压消毒灭菌时会导致快速的分解，所以应该只用最短的时间。

打开高压消毒锅之前要等他完全冷却下来。

甲醛，一氧化碳，二氧化碳
