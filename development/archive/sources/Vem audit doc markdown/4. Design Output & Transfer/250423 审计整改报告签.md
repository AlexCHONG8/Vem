# 管理者代表任命书

为了贯彻执行GB/T42061-2022idtISO13485：2016《医疗器械质量管理体系用于法规的要求》和原国家食品药品监督管理局2014第64号令《医疗器械生产质量管理规范》，加强对质量管理体系运行的领导，特任命杨荣干为本公司的管理者代表。

管理者代表的职责是：

a）贯彻执行医疗器械有关法律、法规、规章和标准等，确保在整个公司内提高满足法规要求、质量管理体系要求和顾客要求的意识。


b）确保将质量管理体系所需的过程形成文件。

c）组织建立和实施与所生产医疗器械相适应的质量管理体系，并保持其科学、合理与有效运行，向最高管理者报告质量管理体系的运行情况和任何改进需求。

d）制定并组织实施公司质量管理体系的审核计划，协助公司最高管理者按计划组织管理评审，编制审核报告并向公司管理层报告评审结果。


e）组织公司内部医疗器械质量管理培训工作，提高员工的质量管理能力。

f）在公司接受各级药品监督管理部门监督检查时，与检查组保持沟通，提供相关信息、资料，配合检查工作；针对检查发现的问题，组织公司相关部门按照要求及时整改。

当公司的生产条件不再符合医疗器械质量管理体系要求，可能影响医疗器械安全，有效时，应当立即向公司最高管理者报告，协助其及时开展停止生产活动、原因调查、产品召回等风险控制措施，并主动向浙江省药品监督管理部门报告。


h）当公司生产的医疗器械发生重大质量问题时，应当立即向公司最高管理者报告，协助其迅速采取风险控制措施，按要求向浙江省药品监督管理部门报告。

i）组织上市后产品质量的信息收集工作，及时向公司最高管理者报告有关产品投诉情况、不良事件监测情况、产品存在的安全隐患，以及接受各级药品监督管理部门监督检查等外部审核中发现的质量管理体系缺陷及其整改情况等。


j）定期组织公司按照《医疗器械生产质量管理规范》要求对质量管理体系运行情况进行全面自查，并于每年年底前向浙江省药品监督管理部门提交年度自查报告。

k）其他法律法规规定的工作。

仓库管理规定（文件编号：QW-PD-03）

1. 目的

本制度对于仓库的收、发、贮、管作了规定，以确保原材料，半成品，成品，退货品的正确入库、出库及库存的准确性和合理性。

2.范围

本制度适用于原材料、半成品、成品、退货品的仓库管理。不适用于中间品和危险化学品及管制化学品。

3.职责

3.1仓库管理员

3.1.1 负责原材料、半成品、成品，退货品的收、发、贮、管工作。根据部门需求，将物料的名称、规格、批号、数量与实物一一核对后办理出、入库手续。遵循先进先出原则进行操作，并确保账、物、卡一致。

3.1.2 负责原材料，退货品到货后的报检，根据QW-QD-01《批号管理制度》，为物料编制相应批号（注：退货品不需要重新编制批号）。

3.1.3负责销售订单中产品出库前的报检。

3.1.4将物料按批次进行管理，按照物料类别、批次分别摆放在指定的仓库、货位上并按照QP16《标识和可追溯性的管理程序》要求做好标识；每月底对库存和账面数量进行盘点，确保及时将临近有效期120天内的物料移至指定区域或清出仓库。

3.1.5负责生产过程中的生产领料和退料。

3.1.6根据需要，参与不合格品的评审活动。

3.1.7 负责仓库门窗、电器、防鼠板检查，切实做好防火、防水防潮、防盗。防虫鼠工作，做好仓库的6S管理工作。

3.2采购部

3.2.1负责通知物料控管物料到达或预计到达情况。

3.2.2负责与供方沟通不合格品的处理事宜。

3.3 质量部

负责原材料、半成品、成品、退货品的检验。验证确认。出具检验报告，方便仓库进行出入库工作。

# 附件53：风险管理计划中增加工具的使用

故障的严重程度、发生概率及可探测度，计算每一个风险的风险水平，输出风险控制措施，确认风险控制措施已在最终设计中实施，并且确保实施的措施确实降低了风险，最终形成《笔试注射器DFMEA》，质量部、专案管理部、生产部、市场销售部、法规注册部负责对其评审；

③市场销售部主导对产品使用过程的风险分析，识别使用过程中可能产生的风险，包括产品后续由客户的装配以及最终的使用过程，经风险分析及风险控制后最终形成《笔试注射器UFMEA》，质量部、专案管理部、研发部、生产部、法规注册部负责对其评审。

④FMEA 是一个动态文件，应该总是反应最终的过程活动。当发生任何过程、设计更改时，各职能应该复审该文件，并根据更改的情况决定是否需要重新研究风险过程和/或更新文件。
