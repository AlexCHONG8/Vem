### Approval Authority Matrix

| Document Type                           | Prepared By           | Reviewed By               | Approved By               |
| --------------------------------------- | --------------------- | ------------------------- | ------------------------- |
| **Quality Procedures (QP)**       | QA Manager            | Management Representative | General Manager           |
| **Standard Procedures (SOP/SIP)** | Department Manager    | QA Manager                | Department Director       |
| **Validation Plans (VMP/VP)**     | RD/Production Manager | QA Manager                | Department Director       |
| **Validation Reports**            | Validation Engineer   | Department Manager        | QA Manager                |
| **Risk Management (RMP)**         | RD Manager            | Management Representative | General Manager           |
| **Biocompatibility (BEP)**        | RD Engineer           | RD Manager                | Management Representative |
| **GSPR Checklist**                | RA Director           | Management Representative | General Manager           |
