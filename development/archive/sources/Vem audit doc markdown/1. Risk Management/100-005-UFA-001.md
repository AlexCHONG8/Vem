## 2. 评分标准（FDA Emergency-Use优化）

### 2.1 发生度 (O) - Emergency Adjusted

| 评分 | 描述 | Emergency Use定义             |
| ---- | ---- | ----------------------------- |
| 5    | 极高 | 紧急情况下失效几乎必然 (>1/2) |
| 4    | 高   | 高压力下频繁失效 (1/3)        |
| 3    | 中等 | Emergency场景偶尔失效 (1/8)   |
| 2    | 低   | 即使紧急情况较少失效 (1/20)   |
| 1    | 极低 | 符合99.999%可靠性 (<1/100)    |

### 2.2 严重度 (S) - Life-Critical

| 评分 | 描述   | Emergency Impact           |
| ---- | ------ | -------------------------- |
| 5    | 极严重 | 立即危及生命，治疗完全失败 |
| 4    | 严重   | 治疗效果大幅降低           |
| 3    | 中等   | 注射延迟或部分失效         |
| 2    | 轻微   | 不影响主要治疗             |
| 1    | 最小   | 无治疗影响                 |

### 2.3 RPN风险分级

| RPN范围 | 风险等级 | FDA合规要求           |
| ------- | -------- | --------------------- |
