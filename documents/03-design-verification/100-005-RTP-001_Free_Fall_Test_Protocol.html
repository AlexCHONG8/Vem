2. Purpose

To verify that Simject series - Epinephrine Pharma Bridge (Adult) project products maintain performance compliance after free fall testing.

3. Scope

This protocol applies to free fall testing for Simject series - Epinephrine Pharma Bridge (Adult) project.

4. Responsibilities

4.1. Analyst: Responsible for drafting this protocol, completing testing according to the protocol, and issuing test reports.

4.2. R&D Engineer: Responsible for reviewing this protocol.

4.3. R&D Director: Responsible for approving this protocol.

9.4. Validation Procedure

9.4.1. Environmental Conditions

Test laboratory temperature and humidity must be controlled within 25 ± 3°C and 55 ± 20% RH range.

9.4.2. Free Fall Preconditioning

9.4.2.1. Use auto-injectors assembled per Section 8 for free fall preconditioning. Note that before free fall preconditioning, the injector cap should never have been removed, maintaining a tightly closed state.

9.4.2.2. Drop injectors from 1000 mm (1 m) height onto the inner surface of the "free fall test box" in the following three drop orientations:

9.4.2.2.1. Horizontal orientation: Drop 10 new injectors in an undisturbed manner. If container breakage is easily observed, do not proceed to next test. Determine as test failure.

9.4.2.2.2. Vertical orientation (rear cap downward): Drop 10 new injectors in an undisturbed manner with rear cap facing downward. If container breakage is easily observed, the system shall not proceed to next test. Determine as test failure.

9.4.2.2.3. Vertical orientation (cap downward): Drop 10 new injectors in an undisturbed manner with cap facing downward. If container breakage is easily observed, the system shall not proceed to next test. Determine as test failure.

9.4.2.3. Each test orientation allows up to three obvious container fractures, otherwise determine as test failure.

9.4.2.4. For each orientation, a total of 10 injectors are dropped, totaling 30 injectors across all orientations.

9.5. Functional Testing

Perform functional testing on the 30 injectors that underwent free fall preconditioning according to 100-005-TIS-023 (V1.0) functional test method.

No. Test Item Test Method
1 Dose accuracy 100-005-TIS-023 (V1.0)
2 Needle extension length (during injection)
3 Cap removal force
4 Injection activation force (pressing needle shield)
5 Injection time for 0.3 mL dose
6 Drug visibility before use
7 Post-injection needle shield lockout force
8 Device status indication
9 Needle shielding
10 Injection start audible prompt
11 Sharps injury protection

9.6. Acceptance Criteria

9.6.1. If complete container breakage is easily observed in any test orientation, determine as test failure.

9.6.2. Each test orientation allows up to three obvious container fractures, otherwise determine as test failure.

9.6.3. Functional test results of auto-injectors after dropping shall meet the standard requirements in the following table:

No. Test Item Acceptance Criteria
1 Dose accuracy 0.27 < injection volume < 0.34 mL
2 Needle extension length (during injection) 0.6 in (±0.1 in) / 15.24 mm (±2.54 mm)
3 Cap removal force 5-35 N
4 Injection activation force (pressing needle shield) 5-25 N
5 Injection time for 0.3 mL dose ≤ 2 seconds
6 Drug visibility before use Window clearly identifiable
7 Post-injection needle shield lockout force >50 N
8 Device status indication PFS plunger visible at window after injection
9 Needle shielding When injector contacts test surface, needle is not visually visible; when injector does not contact test surface, needle does not protrude
10 Injection start audible prompt Measured at 30 cm from device, loudness ≥ 60 dB
11 Sharps injury protection ISO 23908-2024 Appendix test method

9.7. Validation Report

9.7.1. Report shall contain detailed information of used auto-injectors and PFS, including model, batch number, quantity, etc.

9.7.2. Report shall specify the specific process and detailed data of free fall preconditioning and functional testing.

9.7.3. Report shall include result determination for free fall testing. If testing fails, re-testing shall be performed until passed. Each testing process and result shall be recorded and described in detail in the report.
