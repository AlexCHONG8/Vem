# 1. 目的

为了支持用户对用于注射肾上腺素注射液（预充式）的一次性自动注射笔的具体需求，本文档描述了该产品应具有的功能和性能。

# 2. 范围

浙江森迈合创医疗科技有限公司以本文档为基础，按本文档的要求范围来执行产品的设计、开发、验证和制造。

# 3. 缩写及术语

<table><tr><td>缩略词</td><td>英文定义</td><td>中文定义</td></tr><tr><td>URS</td><td>User Requirement Specification</td><td>用户需求规格</td></tr><tr><td>ISO</td><td>International Organization for Standards</td><td>国际标准化组织</td></tr><tr><td>ASTM</td><td>American Society of Testing Materials</td><td>美国材料实验协会</td></tr></table>

# 4. 预期用途

用于搭配用户肾上腺素注射液（预充式）成人剂量  $0.3\mathrm{mg} / 0.3\mathrm{ml}$  、儿童剂量 $0.15\mathrm{mg} / 0.3\mathrm{ml}$ ，进行药液传输（肌肉注射）。仅供单个患者或个人单次使用。

# 5. 适用患者

适用人群：成人款适用于体重大于等于  $30\mathrm{kg}$  的患者，儿童款适用于体重为  $15 - 30\mathrm{kg}$  的患者。
