# 1.产品简介

# 1.1产品名称及型号

产品名称：自动注射笔

产品型号：SM-AISI104、SM-AISI105

产品规格：0.30mL、0.15mL

# 1.2 预期用途及适用人群用途：

# 1.2.1 预期用途

用于搭配肾上腺素注射液（预充式） $(0.3\mathrm{mg} / 0.3\mathrm{mL},0.15\mathrm{mg} / 0.3\mathrm{mL})$ ，进行药液传输（肌肉注射）。仅供单个患者或个人单次使用

# 1.2.2 适用人群：

成人款：体重大于等于  $30\mathrm{kg}$  的患者

儿童款：体重为  $15\sim 30\mathrm{kg}$  的患者

# 4.4风险分析

目的：系统性地识别与医疗器械相关的已知和可预见的风险。

风险管理方法

采用1-5分制对每一项失效模式的发生度(O)、严重度(S)和探测度(D)进行评分，并计算风险优先数（RPN）：  $RPN = O \times S \times D$

RPN ≥ 27：必须采取风险降低措施

RPN<27：可视情况优化，非强制

# 5.总体剩余风险：

评估整体残留风险的方法和整体残留风险的风险可接受性标准在所有风险控制措施实施并验证后，将评估医疗器材造成的整体残留风险，考虑到所有风险的贡献残余风险。

如果相对于预期用途的益处，整体残余风险被认为不可接受，将考虑实施额外的风险。

险控制措施或修改医疗器材或其预期用途。

我们的医疗器械应该完全满足整体残余风险的可接受性标准。

整体残余风险的可接受性标准将由具有应用知识和临床专业知识的专家来判断。

为了定义整体剩余风险的可接受性标准，我们应该考虑以下几个输入和考虑因素：

5.1 不同的事件顺序可能导致不同的危险状况和风险。

5.2 不同的危险状况可能产生特定的危害。

5.3 适用于独立个体风险的风险控制措施可能会导致要求相互冲突，从而增加整体剩余风险。

5.4 警告考虑本身可以充分降低个人风险。然而，过多的警告可能会使用户感到困惑，从而降低单一警告的效果。可能需要进行分析以确定是否过度依赖警告以及这种过度依赖是否会对风险降低和整体剩余风险产生影响。

5.5对医疗器材的所有操作说明进行全面审查可能会发现这些说明不一致或太难遵循。

5.6 设计验证、可用性研究、临床评估、临床前评估和临床调查的结果可以提供有关整体残留风险的有用信息。

5.7 应考虑个别风险的所有效益-风险分析。

5.8 当风险分析中存在风险之间的权衡时，应格外谨慎地分析对整体剩余风险的影响。

5.9 评估总体残余风险的方法可以采取以下途径：

a. 与医疗器材预期用途相关的益处与整体残余风险进行权衡。该评估将考虑适应症的知识、技术和医学领域公认的最新技术水平以及替代医疗设备或治疗的可用性。

b. 残余风险的视觉表示可能很有用。每个单独的残余风险将显示在下列风险矩阵中。如果许多风险位于风险矩阵的较高严重性区域或较高机率区域，则风险的分布可能表明总体剩余风险可能不可接受，即使每个单独的风险已被判断为可接受。

c. 我们的医疗器械与市场上其他同类医疗器械的比较。考虑到预期用途的差异，医疗器材造成的残余风险可以与类似医疗器材的相应风险进行单独比较。

d. 专家支持评估与使用所考虑的医疗器材的预期益处相关的整体残留风险。

e. 尽管此时所有个别风险应已被识别、控制并判断为可接受，但作为整体剩余风险评估的结果，进一步调查某些风险可能是适当的。尤其是那些几乎不可

接受的风险。

f. 当某些风险与其原因或所采用的风险控制措施相互依赖时，进一步调查也是适当的。风险控制的有效性不仅要单独进行，还要与其他风险控制措施结合进行验证。失效分析(FMEA)和事件树分析(ETA)可以成为发现风险和风险控制措施之间关系的有用工具。

# 6.风险控制措施实施及有效性验证：

所有能够降低风险的风险控制措施应经过验证。验证将在风险管理报告中进行总结。风险控制措施的验证可以作为设计/开发验证和流程验证的一部分进行。然而，风险控制措施有效性的验证可以作为设计/开发验证的一部分进行。

# 7.上市后产品资讯收集及审查：

生产后信息将依照产品质量调查）、医疗器材报告、上市后信息收集监测（PMS））、上市后临床追踪（PMCF）符合ISO13485:2016Sec8.2“监测和测量”。

在风险管理过程中，我们利用临床前数据结果来识别器械相关危害并验证风险控制措施的有效性，然后透过不断收集、分析和评估临床数据，包括系统性临床评估中的临床效益来验证安全性设备的性能和性能，作为风险管理报告输入的一部分。
