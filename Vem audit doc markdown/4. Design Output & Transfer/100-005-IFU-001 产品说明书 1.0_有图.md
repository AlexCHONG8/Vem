# （七）禁忌症、注意事项、警示以及提示的内容

# 1、禁忌症：

对肾上腺素或本品辅料有严重过敏史、非过敏性休克的患者（如心源性休克、感染性休克）禁止使用。

禁止静脉内注射，本品专为肌肉注射设计。

由于肾上腺素具有强效的升压作用，高血压危象、未控制的症状性心律失常患者以及器质性脑损伤患者通常禁止使用。

# 2、注意事项、警示以及提示的内容

# $\bullet$  注意事项

患有心血管疾病、脑血管疾病、代谢性疾病、神经系统疾病或正在服用特定药物的患者（ $\beta$  - 受体阻滞剂、单胺氧化酶抑制剂、三环类抗抑郁药、洋地黄类等药物）请谨慎使用。

如果违背下面的指导，可能会带来如下风险，包括错误注射、疾病交叉传播或感染。如果出现上述情况，请立即向您的医生求助。

请遵循本产品的使用说明，并且要在和您的医生和/或专业医护人员交流后才能使用本产品。

未成年人请在监护人、医生的帮助下使用，切勿自行使用。

盲人或视力障碍的病人必须在受过专门培训人员的帮助下使用本产品。听觉或触觉有障碍的病人在使用前请咨询医生。

妥善保管您的产品，要避免儿童或其他不了解正确使用方法的人员的接触。

注射完成以后，请按照当地医疗器械废弃物处理规范进行处理。

# 警告（确保正确应用）

本产品仅用于规定的急救项目。任何更改都必须得到您的经治医生的许可。

每次注射前，请确认您的注射笔中的PFS胶塞位置，并确认笔芯完好无损，且与厂家提供的产品描述相符。

遵循医生和/或专业医护人员的指导，在合适的部位进行注射。

避免将产品放置到极端的温度环境下，包括高温、低温、阳光直射等条件下。

避免产品跌落或接触硬物。请勿将产品浸入水中或任何液体中。

（九）产品维护和保养方法，特殊储存、运输条件、方法：

- 避免阳光直射或骤冷骤热的条件下存放和运输。

- 请勿将本产品浸入水中，否则会导致产品损坏。

- 请小心存放本产品。防止水、灰尘和潮气进入。

- 运输和存放应避免阳光直射，勿靠近热源，请勿冷冻。

- 请勿使用烈性化学制品、洗涤剂或强洗涤剂清洁注射助推器及其配件。

- 请勿对注射助推器及其配件进行蒸汽消毒或者高压消毒。

- 外壳清洁请使用干燥洁净柔软的棉布，或用干燥的棉布蘸取少量医用酒精轻轻擦拭，避免让酒精滴入或流入注射助推器中。

- 请勿使自动注射器及其配件受到强烈的冲击或震动，以免损坏外壳及其内部组件。

- 请勿擅自拆卸注射助推器及其内部组件，否则该设备将不在本公司保修的范围之内。

- 自动注射器存放在儿童不能触及的地方。

- 废旧的自动注射器以及包装材料可能对环境造成污染，请按有关规定回收处理。

- 乘坐飞机时应将产品随身携带，请勿放在行李中托运。

- 自动注射器应存放在阴凉、干燥处，温度范围  $-20^{\circ} \mathrm{C} \sim 55^{\circ} \mathrm{C}$  （未装药物情况下），相对湿度  $20\% \sim 90\%$  ，大气压力  $70 \mathrm{kPa} \sim 106 \mathrm{kPa}$ ，并在相同条件下运输。
