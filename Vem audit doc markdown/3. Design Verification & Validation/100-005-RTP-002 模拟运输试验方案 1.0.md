# 2. 目的

验证新博研桥（成人）专案产品在经过模拟运输试验，仍能保持其性能符合要求。

# 3. 范围

本方案适用于新博研桥（成人）专案模拟运输试验。

# 4. 责任

分析员：负责起草本方案并按照方案完成测试并出具试验报告。

研发工程师：负责审核本方案。

研发负责人：负责批准本方案。

# 6. 定义

- 验收标准：运输单元经受测试计划后必须达到的可接受质量水平。

- 保证等级：基于其在典型流通周期中发生概率的测试强度等级，I级为高强度测试等级，发生概率低；III级为低强度测试等级，但对应发生概率高；II级介于两者之间。

- 流通周期：按顺序列出的测试计划，用于模拟运输单元从生产到消费的特定路径中预期遇到的危害要素

- 运输单元：将经受流通环境的最小完整单元，例如运输容器及其内容物。

# 8. 模拟运输试验程序

# 8.1. 预处理条件

在标准大气条件下（ $23^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C} / 50\% \mathrm{RH} \pm 2\%$ ）进行测试，调节72小时或确保包装系统各组分达到平衡状态。

# 8.6. 委托检验

8.6.1. 模拟运输试验部分由浙江森迈合创医疗科技有限公司（本公司，以下简称“森迈”）委托上海国研医疗器械检测中心有限公司进行。

8.6.2. 上海国研医疗器械检测中心有限公司需提供以下原始数据：振动测试PSD谱图；温湿度曲线记录（1分钟间隔）；跌落冲击波形（峰值加速度、速度变化量）。

8.6.3. 模拟运输实验按照 8.5 测试计划完成后，测试样品由森迈自行取回，以进行后续功能测试。

# 9. 功能测试

# 9.1. 测试环境要求

测试实验室的温度和湿度必须控制在  $25 \pm 3^{\circ} \mathrm{C}$  和  $55 \pm 20 \%$  RH 范围内。

# 9.2. 测试实验室

功能测试部分在森迈实验室内进行测试，使用经过第8节模拟运输试验后取回的前组件和后组件，与批号XX的预灌封注射器（PFS）组装为成品注射笔后进行功能测试。

# 9.5. 自动注射笔组装

使用经过第8节模拟运输试验后取回的前组件和后组件，与批号XX的预灌封注射器（PFS），按照《000-000-TIS-001 Lisa&Lucy 手动终端组装工装测试指导V1.0》的指导进行组装，得到成品自动注射笔。

# 9.6. 测试数量

功能测试共测试30支注射笔。在注射笔组装时，前组件和后组件使用第8节模拟运输试验后取回的前组件和后组件，分别从前组件和后组件包装箱中各抽样30个前、后组件，抽样时请确保每层吸塑托盘中抽取3个，10层吸塑托盘均抽取样品，并保证整个抽样过程随机平均抽样。

# 9.7. NIS测试前环境调节

9.5节组装完成的注射笔。在测试前，需将其放置于温度： $23 \pm 5^{\circ} \mathrm{C}$  和相对湿度： $50 \pm 25 \%$  RH的标准条件下状态调节至少4小时。

# 11. 报告

11.1.1. 报告中应包含使用注射笔和 PFS 的详细信息，包含型号、批号、数量等。

11.1.2. 报告中应注明模拟运输试验（包含功能测试）的具体过程和详细数据等。
