# 3. 产品描述

# 3.1 产品名称及型号

产品名称：自动注射笔

# 3.2预期用途及适用人群：

# 3.2.1 预期用途：

欣捷达自动注射笔适用于搭配肾上腺素(预充式0.3mL)进行药液传输(肌肉、注射)仅供单个患者或个人使用。

# 3.2.2 适用人群：

用于抢救紧急过敏反应(如过敏性休克，哮喘)、心脏骤停等的患者（未成年人和成年人）。注射部位存在感染、炎症或皮肤破损者禁用。

# 3.3产品组成结构

欣捷达自动注射笔由前组件、后组件及笔帽组件组成，其中后组件的注射助推器由注射器载体和动力弹簧装置两部分组成。

# 3.6货架寿命和运输储存对生物相容性影响的评价

欣捷达自动注射笔运输储存条件：

1. 储存条件：温度： $5^{\circ} \mathrm{C} \sim 40^{\circ} \mathrm{C}$ ；湿度  $\leqslant 80\%$

2. 运输方式：空运或陆运

欣捷达自动注射笔生物相容性不会受到货架寿命的影响。

理由：欣捷达自动注射笔产品使用的高分子材料PC+ABS、PC等材料在正常使用条件下不易降解。基于其物理特性，欣捷达自动注射笔中高分子材料的生物相容性不会受到货架寿命的影响。

# 5. 材料的安全性研究

# 5.1.PC

Polycarbonate（聚碳酸酯，简称PC）是一种高性能的热塑性塑料，具有优异的物理和化学特性，广泛应用于工业、电子、医疗、建筑等领域，高强度抗冲击性强，长期使用温度可达 $120^{\circ}\mathrm{C}$ ，短期可耐  $140^{\circ}\mathrm{C}$  以上，但易被有机溶剂（如丙酮、强碱）腐蚀。长期暴露在紫外线下会变黄。其细胞毒性、致敏性和刺激性测试（如MTT法、皮内反应试验）结果符合ISO10993-5、-10、-23的要求。通常生物相容性良好，但需关注残留单体双酚A（BPA）及可能水解降解的风险。

# 5.2.PC+ABS

聚碳酸酯-丙烯腈-丁二烯-苯乙烯共聚物(PC+ABS)结合了聚碳酸酯(PC)的强度、耐热性和ABS的韧性，广泛应用于医疗器械(如外壳、手柄、呼吸设备等)，其细胞毒性、致敏性和刺激性测试(如MTT法、皮内反应试验)结果符合ISO10993-5、-10、-23的要求。通常生物相容性良好，但需关注残留单体（丙烯腈、双酚A）可能影响结果的风险。

PC+ABS在短期接触器械中应用广泛，适用于非植入器械（如设备外壳、手柄等、呼吸面罩等），PC+ABS已成功用于胰岛素笔、雾化器外壳等低风险器械的上市，其预期的生物学反应水平是可接受的。
