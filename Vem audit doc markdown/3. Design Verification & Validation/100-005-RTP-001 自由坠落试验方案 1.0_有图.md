# 9.4. 验证程序

# 9.4.1. 环境条件要求

测试实验室的温度和湿度必须控制在  $25 \pm 3^{\circ} \mathrm{C}$  和  $55 \pm 20 \%$  RH 范围内。

# 9.4.2. 自由坠落预处理

9.4.2.1. 取8.节组装完成的自动注射笔进行自由坠落预处理，注意自由坠落预处理前注射笔保持笔帽从未拔开，保持紧闭的状态。

9.4.2.2. 按以下三个坠落方向将注射笔从  $1000 \mathrm{~mm}$ （ $1 \mathrm{~m}$ ）高度自由坠落到“自由坠落试验箱”内表面。

9.4.2.2.1. 水平以无扰的方式坠落 10 个新的 注射笔; 如果很容易地发现容器完全破碎, 则不再进行下一步试验。判未通过该试验。

9.4.2.2.2．垂直以后盖向下方向，以无干扰的方式坠落10个新的注射笔，如果很容易地发现容

器完全破碎,则该系统不再进行下一步试验。判未通过该试验。

9.4.2.2.3.垂直以笔帽向下方向，以无干扰的方式坠落10个新的注射笔，如果很容易地发现容

器完全破碎,则该系统不再进行下一步试验。判未通过该试验。

9.4.2.3. 每个试验方向允许出现三个明显容器破裂,否则判未通过该试验。

9.4.2.4. 对每个朝向累计投下的10支注射笔，共计30支注射笔。
