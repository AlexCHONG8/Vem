# 8.1.3. 测试环境要求

温度:  $23 \pm 5^{\circ} \mathrm{C}$  和相对湿度:  $50 \pm 25 \% \mathrm{RH}$

# 8.1.4. 注射笔测试前环境调节

在测试前，需在温度：  $23\pm 5^{\circ}C$  和相对湿度：  $50\pm 25\% \mathrm{RH}$  的标准条件下状态调节至少4小时。

# 8.1.5. 自动注射笔组装

将调节好的前组件和后组件，与批号 XX 的预灌封注射器（PFS），按照《000-000-TIS-001 Lisa&Lucy手动终端组装工装测试指导 V1.0》的指导进行组装，得到成品自动注射笔。

# 8.1.6. 测试方法

请按照《100-005-TIS-023 V1.0总装》进行三个独立批次，每个批次样品10支进行功能测试。
